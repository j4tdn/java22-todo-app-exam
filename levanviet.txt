4.Phân biệt git merge và git rebase(giống, khác nhau, khi nào nên sử dụng)

giống : đều là gộp nhánh
khác :  + git merge thêm các thay đổi từ nhánh này sang nhánh khác
        + git rebase chuyển các thay đổi từ một nhánh sang 1 nhánh khác 

5.Phân biệt git fetch và git clone
giống : đều dùng để tải code từ git về máy để làm
khác :  +git clone : lấy cả về máy
        +git fetch : trọn lọc và có các nhánh trung gian
cách dùng : +git clone:   dùng khi bắt đầu vào dự án 
             +git fetch : dùng khi up code lên git 

6.Phân biệt git fetch và git pull

giống : Tải xuống các thay đổi từ một kho lưu trữ từ xa
khác + git pull : tải xuống và gộp thành 1 nhánh
