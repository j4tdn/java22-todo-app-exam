4, Giống: Đều dùng để gộp nhánh, có kết quả giống nhau, tương tự khi gộp nhánh chung đường thẳng
   Khác: Khi gộp nhánh rẽ hướng
   Merge: Gộp nội dung từ các commit mới từ nhánh cha, chỉ cần fix 1 lần khi bị conflict, tạo commit mới( commit merge)
   Rebase: Gộp lần lượt các commit, có thể nhiều conflicts, commit gộp thay đổi mã hash, không tạo commit mới

5, Git clone: sao chép tất cả remote repo về máy local
   Git fetch: cập nhật nội dung trên remote repo về máy local

6, Git fetch: cập nhật nội dung trên remote repo về máy local
   Git pull: vừa cập nhật nội dung từ remote vừa cập nhật local repo tương ứng  
   Git pull= git fetch + git merge

7, Git pull= Git fetch + git merge 
   Git pull --rebase= Git fetch + git rebase
 Khác: git pull tạo 1 commit merge mới 
       git pull --rebase: không có commit mới

18, Các lệnh xóa commit: 
git reset --hard HEAD~1: xóa commit, xóa luôn nội dung
git reset --soft HEAD~1: xóa commit, giữ nội dung

19, Các trạng thái của file trong git:
Untracked: Các file chưa add, chưa commit
Stage: Các file đã được thêm thành công, chuẩn bị commit
Modifiled: File đã được sưả đổi từ file đã được commit
Unmodifiled:File đã được commit thành công
